What is Mathematica?

Mathematica is a symbolic mathematical computation program, sometimes called a computer algebra program, used in many scientific, engineering, mathematical, and computing fields. It was conceived by Stephen Wolfram and is developed by Wolfram Research of Champaign, Illinois. The Wolfram Language is the programming language used in Mathematica.

Using Mathematica

Mathematica uses what is called a notebook as a text editor, which includes a cell hierarchy. This kind of text editor allows for the user to embed their code within plain text, to create a dynamic experience. Hitting Shift+Enter, allows the user toevaluate any written code.

Features and Examples

Arithmetic
Plus[x, y, z, ...], +
    represents a sum of terms.
Times[x, y, z, ...], *
    represents a product of terms.
Power[x, y,], ^
    gives x to the power y.
Conversions
N[expr]
    gives the numerical value of expr.
N[expr, n]
    attempts to give a result with n‐digit precision.
BaseForm[expr, n]
    prints with the numbers in expr given in base n.

Range
Range[imax]
    generates the list {1, 2, ..., imax}.
Range[imin, imax]
    generates the list {imin, ..., imax}.
Range[imin, imax, di]
    uses step di.
Table
Table[expr, n]
    generates n copies of expr.
Table[expr,{i, (imin,) imax (, di)}]
    generates a list of the values of expr when i runs from 1(imin) to imax, using steps di.
Indexing
list[[ i ]]
    picks the ith element from list (i starts at 1).
list[[ -i ]]
    starts from the end.
list[[ i ;; j ]]
    generates a sublist from i to j, inclusive.
Statistics
Mean[list]
Max[list]
StandardDeviation[list]

Assigning
var = val
    assigns val to var.
var := expr
    assigns delayed value of expr (usually a function) to var. When var appears, it is replaced by expr, evaluated afresh each time.
Clearing
var = .
    clears val.
Clear[var]
    clears val.

Defining
f[x_, y_, ...] := expr
f := Function[{x, y, ...}, expr]
    defines a function f in terms of x, y, ....
Solving Equations
Solve[expr, vars, (dom)]
    attempts to solve the system expr of equations or inequalities for the variables vars over the domain dom (Reals, Integers, Complexes).

Plotting Functions
Plot[f, dom]
Plot[{f1, f2, ... }, dom]
    generates a plot of f as a function of x in the domain dom.
Plot3D[f, domX, domY]
Plot3D[{f1, f2, ... }, domX, domY]
    generates a three-dimensional plot of f as a function of x and y in their respective domains.
PolarPlot[r, domθ]
PolarPlot[{r1, r2, ... }, domθ]
    generates a polar plot of a curve with radius r as a function of angle θ.

Limits
(Min/Max)Limit[f, x -> val]
    gives the (lower/upper) limit of f as x approaches a given value val.
Differentiation
f'[x]
    represents the derivative of a function f with respect to variable x.
D[f, {x, n}]
    represents the nth derivative of a function f with respect to x.
Integration
Integrate[f, x]
    gives the indefinite integral ∫fdx.
Integrate[f, dom]
    gives the definite integral ∫domfdx.

Entities
EntityList[class]
     entities in a class
EntityValue[entities,property]
     value of a property of an entity

Maps
GeoDistance[entity1,entity2]
     geo distance between entities
GeoPosition[entity]
     geo position of an entity
GeoNearest["type",location,n]
     nearest n objects of a certain type to a location
Positions
GeoListPlot[entity1,entity2, ...}]
    plot a list of entities on a map
GeoGraphics[entity]
    map constructed from primitives
GeoPath[{entity1,entity2}]
    path between entities
